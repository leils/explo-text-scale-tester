# Scalable Text Height Test

Quick HTML/CSS website to test Explo font weights at various viewport height (vh) percentages.

## Purpose

This tool helps determine appropriate font sizes for 32" touchscreen installations by displaying all font weights at different vh percentages.

## Usage

1. Open `index.html` in a browser
2. View on your target screen size (32" touchscreen)
3. Adjust vh percentages in the HTML to find optimal sizes

## Current Test Range

- **2vh** - Very small text
- **3vh** - Small text
- **4vh** - Medium text
- **5vh** - Large text
- **6vh** - Extra large text
- **8vh** - Very large (headings only)

## Font Weights Tested

- Thin (100)
- Light (300)
- Regular (400)
- Medium (500)
- Bold (700)
- Black (900)
- Ultra (950)

## Notes

- Fonts are imported from the lightweight-ui-library
- All sizes use viewport height (vh) units for scalability
- Adjust percentages in `index.html` to test different sizes
